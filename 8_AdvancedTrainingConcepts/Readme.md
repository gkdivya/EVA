The code is fully modularised, we have built a repo named [torch_cv_wrapper](https://github.com/gkdivya/torch_cv_wrapper) for this assignment, and all the libraries are imported from this:

    |   ├── load_data.py
    ├── model

    ├── utils

    |   ├── train.py 
    |   ├── test.py
